How can you use SPS?





Anyone with a device having browser and internet can easily use Stroke Prediction System. If you want to predict Stroke disease using SPS, all you need is to enter accurate value of 16 different attributes that system asks like age, hypertension, blood pressure and so on. SPS will then return the chances of having Stroke disease in probability which resides from 0-1. In addition to that, if you want to save data for future purpose and want guidance on what to do next, you can simply sign up to the system which provides all those features.
